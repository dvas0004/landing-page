# David Vassallo — Personal Landing Page

Built with Vue 3 + Vite + TypeScript.

## Sections

- **Hero** — Introduction & key roles
- **About** — Bio & education
- **Experience** — Career timeline
- **Skills** — Technical skills by category
- **Certifications** — Security & professional certifications
- **GitHub** — Notable repositories
- **Contact** — Email, LinkedIn, GitHub

## Getting Started

```bash
# Install dependencies
yarn install

# Run dev server
yarn dev

# Build for production
yarn build
```

## Customization

Edit `src/data/profile.ts` to update personal information.

## Deploy

Deployed on Netlify. Push to deploy branch to trigger automatic deployment.
